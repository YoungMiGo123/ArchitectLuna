# CLAUDE.md

Read `AGENTS.md` — it is the single source of truth for working in this repo (project overview,
build/test commands, invariants, and the required feature workflow).

For any non-trivial change, follow `docs/workflow/feature-workflow.md`: classify S/M/L, and for
M/L produce a plan (L → a plan document in `docs/plans/`) **before** implementing. The `/feature`
skill automates this end to end — prefer it for feature requests.
